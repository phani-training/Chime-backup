Amazon Chime Signaling SDK 0.1.0
Amazon Chime Signaling Client is a project that allows builders to use their custom webrtc, but use Amazon Chime signaling server. Signaling server is used to communicate information about audio and video between peers, so the signaling client will enable builders to customize their own webrtc library, but use the Amazon Chime backend.

9// Runnable is an abstraction that allows an implementation to inform the owning object
10// of the polling model used. If polling is required, IsPollable() should be overriden and return true
11// and Poll() should be overridden with polling logic.

15 // It is expected that this call is non-blocking.
16 virtual void Run() {
17 if (running_) return;
18 running_ = true;
19 run_thread_ = std::thread([=]() {
20 while (running_) {
21 Poll();
22 }
23 });
24 }

26 virtual void StopRun() {

39 private:
